StockScan360

A smart inventory and barcode management system built entirely with HTML, CSS, and JavaScript.
Features

    Barcode Scanning: Uses html5-qrcode to scan barcodes directly via the device camera.
    Inventory Management: Add, edit, and delete items with real-time updates.
    Analytics Dashboard: Visualizes inventory data by category and calculates total stock value.
    Data Persistence: Uses localStorage to save data in the browser.
    Responsive Design: Works smoothly on desktops, tablets, and mobile phones.
